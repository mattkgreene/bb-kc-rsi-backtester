# UI Style Notes

Direction: light, modern analytics dashboard with warm neutrals and bright accent.

Tokens:
- Background: #f4f1eb with warm gradients
- Panels: #ffffff / #f8f5ef
- Accent: #ff6b3d
- Secondary accent: #14b8a6
- Text: #1d2430

Layout:
- Fixed sidebar with card-like sections
- Main panel with tabs and responsive metric grid
- Rounded buttons and inputs
